HTML

Esta etiqueta envuelve todo el contenido del documento HTML. Es la etiqueta de apertura y cierre que indica el inicio y fin del documento HTML.

HEAD

Esta etiqueta contiene la información de encabezado del documento HTML.

BODY

Esta etiqueta contiene todo el contenido visible del documento HTML, como texto, imágenes, enlaces, etc. Lo que incluyas dentro de estas etiquetas se mostrará en la ventana del navegador
